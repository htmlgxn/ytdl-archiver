# v0.3.0 Release Notes

## Summary
`v0.3.0` is a comprehensive release-readiness update focused on behavior clarity, documentation parity, and metadata-sidecar/NFO pipeline coverage.

## User-visible highlights
- CLI docs now cover all first-class commands:
  - `archive`
  - `metadata-backfill`
  - `search`
  - `convert-playlists`
  - `init`
- Verbose mode contract clarified:
  - structured diagnostics only
  - no raw yt-dlp passthrough output
- Metadata artifact contract documented as default:
  - `<stem>.info.json`
  - `<stem>.metadata.json`
  - enriched per-video `.nfo` when enabled
- Subtitle behavior documented with explicit status semantics and sidecar naming rules.
